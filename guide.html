모비페이 POS(브라우저) 설정 매뉴얼

안녕하세요! 모비페이 차량번호 인식 테스트 페이지 입니다.

현재 모비페이는 개발중으로, 100% 완성되지는 않았으나, 테스트 해볼 수 있는 페이지 입니다.

PC의 경우, 인식 테스트에 앞서 하드웨어 가속을 반드시 활성화 해주세요!
PC의 경우, 크롬 확장프로그램으로 정상작동하지 않을 수 있으니, 반드시 시크릿 모드로 접속해주세요!

테스트 페이지 접속 링크 / QR코드는 맨 아래에 있습니다. :)


하드웨어 가속이란?

하드웨어 가속은 이미지 처리 작업을 GPU에 맡기는 기능으로, 차량번호 인식을 위해 반드시 필요하며, 아래 내용은 Google Chrome 기준으로 하드웨어 가속을 활성화 하는 방법입니다. Edge나 Whale도 비슷한 절차로 가능하니 참고해주세요!

하드웨어 가속 설정 방법

  • 가장 빠른 방법: 여기를 눌러 설정페이지로 이동합니다.
  1. Google Chrome을 엽니다.
  2. 우측 상단에 있는 세로 점 세 개 아이콘(더보기 버튼)을 클릭합니다.
  3. 드롭다운 메뉴에서 설정을 클릭합니다.
  4. 설정 페이지 하단으로 스크롤하고, 고급을 클릭합니다.
  5. 시스템 섹션에서 "가능한 경우 하드웨어 가속 사용" 옵션을 켭니다.
  6. Chrome을 반드시 재시작 해주세요! 활성화를 위해 반드시 필요합니다 :)
  • 설정창에서 '가속'을 검색해서 활성화도 가능합니다.
